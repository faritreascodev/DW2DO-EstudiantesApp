# Actividad: AppEstudiantes
Aplicación Android para gestionar información de estudiantes con Kotlin y Jetpack Compose.

## Descripción
AppEstudiantes es una app móvil que permite ver una lista de estudiantes en tarjetas personalizadas y registrar nuevos estudiantes mediante un formulario. Los datos se comparten entre pantallas usando el patrón Singleton con listas mutables.

## Qué se usó
- Kotlin
- Jetpack Compose
- Material Design
- Emulador Android SDK API 21

## Características

- Visualización de estudiantes en Cards con foto, nombre y edad
- Registro de nuevos estudiantes
- Navegación entre Activities
- Diseño responsive con Compose
- Lista mutable compartida globalmente

## Pruebas
1. Clonar el repositorio:
git clone https://github.com/faritreascodev/AppEstudiantes.git

3. Abrir el proyecto en Android Studio

4. Ejecutar en emulador o dispositivo físico

## Cómo usar
1. Desde el menú principal, selecciona "Ver Estudiantes" para visualizar la lista
2. Selecciona "Registrar Estudiante" para agregar uno nuevo
3. Completa los campos y dale "Guardar"
4. Regresa a la lista para verificar el nuevo estudiante
